
# Code of Conduct

## Our Commitment

In the interest of fostering an open, respectful, and collaborative environment, contributors and maintainers of this project pledge to make participation in this community a harassment-free experience for everyone.

We are committed to creating an environment where technical discussion, collaboration, and learning can happen constructively.

---

## Expected Behavior

Examples of behavior that contribute to a positive environment include:

- being respectful and professional
- giving constructive feedback
- being open to different technical viewpoints
- focusing on what is best for the project
- showing empathy toward other contributors
- communicating clearly and in good faith

---

## Unacceptable Behavior

Examples of unacceptable behavior include:

- harassment, abuse, or discriminatory language
- personal attacks or insulting remarks
- trolling or deliberate disruption
- publishing others' private information without permission
- hostile or dismissive behavior that damages collaboration

---

## Scope

This Code of Conduct applies within all project spaces, including:

- GitHub issues
- pull requests
- discussions
- review comments
- project-related community interactions

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing these standards.

Maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that do not align with this Code of Conduct.

---

## Reporting

If you experience or witness unacceptable behavior, please report it privately to the project maintainer.

For now, reports may be directed through GitHub by contacting the maintainer of this repository.

---

## Enforcement Guidelines

Possible responses to unacceptable behavior may include:

- warning the contributor
- requesting behavioral changes
- temporary restriction from participation
- permanent removal from the project space in severe cases

---

## Final Note

This Code of Conduct is intended to support a professional and respectful open-source environment for everyone participating in the project.
